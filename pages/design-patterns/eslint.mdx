# ESLint

O [ESLint](https://eslint.org/) é um code analyst usado para manter padrões, corrigir problemas e configurar o projeto.

Temos o arquivo de configuração main em `.eslintrc.js`, onde definimos seu `parser`, `plugins`, `rules`, entre outras configurações importantes para manter sempre o time codando no mesmo estilo.

Esse arquivo é global é todos os apps e packages utilizam as regras dele.
