# Prettier

O [Prettier](https://prettier.io/) é um code formatter usado para manter um estilo no código, indentação e padrão de escrita.

Temos o arquivo de configuração main em `prettier.config.js`, onde definimos seu `trailingComma` e `singleQuote`, podendo definir outras configurações importantes para manter sempre o time codando no mesmo estilo.

Esse arquivo é global é todos os apps e packages utilizam as regras dele.
