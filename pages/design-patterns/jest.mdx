# Jest

O [Jest](https://jestjs.io/pt-BR/) é um framework para testes usado para manter a segurança e o código funcionando como o esperado.

Temos o arquivo de configuração main em `test/jest.config.base.js`, onde definimos seu `cacheDirectory`, `transformers`, `reporters`, entre outras configurações importantes para manter sempre o código seguro e executando testes rapidamente.

Alem disso, cada app deve ter seu `jest.config.js` extendendo, no mínimo, da configuração base, e colocando as demais desejadas configurações.
