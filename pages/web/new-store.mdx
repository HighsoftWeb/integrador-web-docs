# Adicionando uma nova Loja

Para adicionar uma nova Loja, você precisa ja ter os seguintes dados:
- Plataforma;

Após certificar-se de ter esses dois dados ja criados, você pode seguir criando a Loja.

Para criar a Plataforma, vá para a página de gerenciamento `/admin` e navegue até a opção de Lojas.

Após isso, clique sobre o icone de adicionar '+' no final da opção.

![new-store-plus](/web/new-store-plus.png)

Basta então preencher os campos obrigatórios para cadastrar uma nova loja, clicar em 'Save' e pronto, sua loja estará criada.

![new-store-form](/web/new-store-form.png)
