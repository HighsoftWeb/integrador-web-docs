# Adicionando uma nova Setting

Para adicionar uma nova Setting, você precisa ja ter os seguintes dados:
- Plataforma;
- Loja;

Após certificar-se de ter esses dois dados ja criados, você pode seguir criando a Setting.

Para criar a Setting, vá para a página principal `/` e navegue até a Loja que deseja adicionar uma nova configuração.

Após isso, clique sobre o icone de adicionar '+'.

![new-setting-plus](/web/new-setting-plus.png)

Basta então selecionar uma das plataformas ja cadastradas e pronto, sua configuração estará criada.

![new-setting-options](/web/new-setting-options.png)
