# Scripts

Os scripts são comandos definidos para rodar outros comandos.

Temos os seguintes comandos no `package.json`:
- **build**: faz o build do projeto, usado para fazer deploy;
- **dev**: roda todo o projeto em development mode;
- **web**: roda apenas o projeto `web` em development mode;
- **lint**: roda o eslint em todo o projeto;
- **lint:staged**: roda o eslint para os arquivos alterados;
- **format**: roda o prettier no código;
- **prepare**: roda o husky a cada comando;
